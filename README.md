# PopClipに以下の機能を追加
+ Weblio
+ 英辞郎
+ Urban
+ Farlex

## ダウンロード
[Download ZIP](https://github.com/ShingoFukuyama/my-PopClip-extension/archive/master.zip)
解凍後、中にあるpopclipextで終わるファイルをダブルクリックして開きます。

表示されるウィンドウの左側のインストールボタンをクリックすると反映されます。
